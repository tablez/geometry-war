# 几何战争 (Geometry War)

> 极简几何风格的卡牌构筑 Roguelike 游戏 - 小程序版

## 项目简介

《几何战争》使用 uni-app + Vue3 开发，支持发布到：
- 微信小程序
- 支付宝小程序
- 抖音小程序
- H5
- App

## 技术栈

- **框架**: uni-app 3 (Vue 3)
- **语言**: JavaScript / TypeScript
- **样式**: CSS / SCSS
- **开发工具**: HBuilderX

## 项目结构

```
geometry-war/
├── pages/              # 页面
│   ├── index/          # 首页
│   ├── game/           # 游戏页面
│   └── cards/          # 卡牌图鉴
├── components/         # 组件
├── static/             # 静态资源
├── store/              # 状态管理
├── utils/              # 工具函数
├── App.vue             # 根组件
├── main.js             # 入口文件
├── pages.json          # 页面配置
├── manifest.json       # 应用配置
└── README.md
```

## 开发环境

1. 下载 HBuilderX: https://www.dcloud.io/hbuilderx.html
2. 导入项目
3. 运行到浏览器/微信开发者工具

## 发布

```bash
# 微信小程序
HBuilderX → 发行 → 小程序-微信

# H5
HBuilderX → 发行 → 网站-H5
```
